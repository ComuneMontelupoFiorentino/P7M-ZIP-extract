# P7M-ZIP-extract
Dato in input una cartella compressa, estrae il contenuto e tutti i file `.p7m` vengono estratti in pdf
## Utilizzo
- inserire nella cartella 'input' la cartella compressa `.zip` contenente i file `.p7m`.
- Avviare il programma con doppio click su `Extractor`
- Attendi fino all'apertura del popoup che ti mostrerà il contenuto in input
- Seleziona una o più cartelle zippate e clicca su invia
- Attendi fino a quando il messaggio "Tutte le estrazioni completate!"
- Nella cartella output troverai la cartella decompressa contenente tutti i file e una sottocartella pdf_from_signed contenente i file pdf estratti dai p7m
